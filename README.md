📋 Sobre o Projeto

Esse meu projeto foi baseado na minha empresa de criação de canecas personalizadas, para criar orçamentos para enviar para o cliente final.

✨ Funcionalidades
* **Funcionalidade 1:** Adicionar Imagem do produto pelo código da imagem na pasta raiz ex: ca001 (Caneca 01).
* **Funcionalidade 2:** No Orçamento, mostrará horário que foi criado o orçamento.
* **Funcionalidade 3:** Gera o PDF do orçamento com Logo da Empresa, e dados do cliente contidos...
